## Solution

Language: **java**  /  Runtime: 3 ms

```java
class Solution {
    public static int removeDuplicates(int[] nums) {
        int len=nums.length;
        int index=0;
        int count=0;
        for(int i=0;i<len;i=index) {
            for(int j=i+1;j<len;j++) {
                if (nums[i]==nums[j]) {
                    index++;
                }else {
                    break;
                }
            }
            nums[count++]=nums[index++];
        }
        return count;
    }
}
```
