## Solution

Language: **java**  /  Runtime: 2 ms

```java
class Solution {
    public int missingNumber(int[] nums) {
//        Arrays.sort(nums);
        int sum=0;
        for(int i=0;i<nums.length;i++) {
            sum+=nums[i];
        }
        return (0+nums.length)*(nums.length+1)/2-sum;
    }
}
```
