## Solution

Language: **java**  /  Runtime: 1 ms

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int len=nums.length;
        for(int i=0;i<len;i++){
            if(nums[i]==val){
                len--;
                for(int j=i+1;j<nums.length;j++){
                    nums[j-1]=nums[j];
                }
                i--;
            }
        }
        return len;
    }
}
```
