## Solution

Language: **python3**  /  Runtime: 40 ms

```python3
class Solution:
    def findTheDifference(self, s: str, t: str) -> str:
        words1={}
        words2={}
        for num in s:
            words1[num]=words1.get(num,0)+1
        for num in t:
            words2[num]=words2.get(num,0)+1
        print(words1)
        print(words2)
        for word in words2:
            if word in words1 and words1[word]!=words2[word]:
                return word
            elif word not in words1:
                return word




```
