## Solution

Language: **java**  /  Runtime: 3 ms

```java
class Solution {
    public int search(int[] nums, int target) {
        for(int i=0;i<nums.length;i++) {
            if (nums[i]==target) {
                return i;
            }
        }
        return -1;
    }
}
```
