## Solution

Language: **java**  /  Runtime: 0 ms

```java
class Solution {
    public void rotate(int[][] matrix) {
        int m=matrix.length;

        for(int i=0;i<m;i++){
            for(int j=0;j<i;j++){
                int temp=matrix[i][j];
                matrix[i][j]=matrix[j][i];
                matrix[j][i]=temp;
                // System.out.println(matrix[i][j]);
            }
        }
        for(int i=0;i<m;i++){
            for(int j=0;j<m/2;j++){
                int temp=matrix[i][j];
                matrix[i][j]=matrix[i][m-j-1];
                matrix[i][m-j-1]=temp;
            }
        }
    }
}
```
