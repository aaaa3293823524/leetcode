## Solution

Language: **java**  /  Runtime: 0 ms

```java
class Solution {
// a < b:
// 1 + (a - b) >> 31 == 0
// a > b:
// 1 + (a - b) >> 31 == 1
//public:
    public int maximum(int a, int b) {
        long c = a, d = b;
        int k = (int)(1+((c - d) >> 63));
        return k * a + (1-k) * b;
    }
};


```
