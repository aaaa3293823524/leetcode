## Solution

Language: **java**  /  Runtime: 1 ms

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int[]dp=new int[nums.length];
        dp[0]=nums[0];
        int ans=dp[0];
        for(int i=1;i<nums.length;i++) {
            dp[i]=Math.max(dp[i-1]+nums[i], nums[i]);
            ans=Math.max(dp[i], ans);
        }
        return ans;
    }
}
```
