## Solution

Language: **java**  /  Runtime: 0 ms

```java
class Solution {
    public int missingNumber(int[] nums) {

        int num[]=new int[nums.length+1];
        int sum=0;
        for(int i=0;i<nums.length;i++){
            num[i]=i;
            sum=sum^num[i]^nums[i];
        }
        return sum^nums.length;
    }
}
```
